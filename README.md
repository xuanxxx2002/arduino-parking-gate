# 🚗 Arduino Automatic Parking Gate

## 📌 專案簡介
本專案為一套基於 Arduino 的自動停車柵欄系統，透過超音波感測器偵測車輛距離，自動控制伺服馬達開啟或關閉柵欄，模擬智慧停車場的入口控制機制。

---

## 🎯 專案特色
- 🚘 車輛自動偵測（超音波感測）
- 🔄 即時控制柵欄開關（伺服馬達）
- ⚡ 低成本、易實作
- 🔧 可擴展（RFID / IoT / AI）

---

## 🧩 系統架構
Arduino 透過超音波感測器（HC-SR04）測量距離，當偵測到車輛接近時，觸發伺服馬達開啟柵欄，車輛離開後自動關閉。

---

## 🔌 硬體需求
- Arduino Uno
- HC-SR04 超音波感測器
- SG90 伺服馬達
- 麵包板
- 杜邦線

---

## 🔗 接線方式

### 📡 超音波感測器
| 腳位 | Arduino |
|------|--------|
| VCC  | 5V     |
| GND  | GND    |
| Trig | D9     |
| Echo | D10    |

### ⚙️ 伺服馬達
| 腳位 | Arduino |
|------|--------|
| 紅   | 5V     |
| 棕   | GND    |
| 橙   | D6     |
